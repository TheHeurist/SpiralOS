# Glossary Entrypoint

## Core utterance

The packet should not require immediate immersion in the full vocabulary field.

It should instead provide a lawful threshold into the most load-bearing terms,
so that readers can begin with clarity and descend later with discipline.

This note is that threshold.

## Governing question

### Characteristica Universalis

The note `CHARACTERISTICA-UNIVERSALIS.md` answers the question:

**What do all things have in common?**

At the packet level, CU names the signature-bearing commonality through which
meaning and knowledge are arranged across holon and holarchy.
