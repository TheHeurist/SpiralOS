# HC IX Glossary Canon Charter (v0.9.2)

Date: 2026-02-23

## 1. Purpose

This charter defines how we stabilize, canonize, and present the HC IX glossary:

- comprehensive coverage of `/docs` concepts (with explicit disclosure of limits),
- rigorous formalization where the repo provides it (equations, operators, symbols),
- non-collapsing plurality: when multiple definitions exist, we preserve variants.

## 2. Scope & provenance (panel-facing)

**In scope**

- all Markdown under `docs/**`
- PDFs under `docs/**` only if there is no sibling `.md` with the same filename stem in the same folder

**Evidence format**
Each entry carries at least one source pointer:
`path | heading | lines a-b`

## 2.5 Toolshed Principle (Along/At)

When interiority is relevant, HC IX requires **Along/At Conjugation**:

- an exterior account (**looking at**) and an interior account (**looking along**) must be braided via `⋈`,
- closure uses `ℒ` with a return test (one re-query),
- no automatic “debunking by stance”.

## 3. Canon tiers

**Tier A — Canon**
Single explicit definition → publish verbatim (minor punctuation only).

**Tier B — Canon with Variants (collision)**
Multiple explicit definitions → publish preferred HC IX meaning + list variants with sources; no silent merges.

**Tier C — Needs Definition (gap)**
Used but not explicitly defined → mark `NEEDS_DEFINITION` + top sources; optional editorial working definition clearly labeled.

## 4. Canon selection rules

Priority for preferred HC IX meaning:

1) Holor Calculus pillars / formal appendices
2) explicit glossary sections
3) volume appendices with explicit definitions
4) narrative/offerings/responses

## 5. Symbols and equations integrity

- Every symbol used in an HC IX-facing equation must appear in the symbol table.
- Threshold overload is disambiguated: reserve `τ` for Spiral Time (introduced in HC IX); use `τ_c` for thresholds.

## 6. Change management

All canon decisions are logged:
`Term | change type | decision | rationale | date`
